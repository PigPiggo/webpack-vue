# 初始化项目
## 1. init项目
- 我用的yarn 你也可以用npm
- 这步很简答，一路点过来就行
- entry是指入口文件修改问main.js
## 2. 基础依赖
1. webpack相关
    - `yarn add webpack webpack-cli webpack-dev-server -D`
    - 安装webpack、webpack-cli、webpack-dev-server
    - 注意我用的版本号
    - 这里一定不要忘记 -D
2. vue相关
    - `yarn add vue-loader vue-template-compiler -D`
    - 详见[介绍 | Vue Loader](https://vue-loader.vuejs.org/zh/)
    - `yarn add vue`
    - 安装vue.js
3. 新建项目结构
    - -build: 打包及相关
    - -config: 项目配置
    - -src: 业务代码
